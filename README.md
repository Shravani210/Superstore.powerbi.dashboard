<h1 align="center">📊 Superstore Sales Dashboard (Power BI)</h1>

## 📌 Project Overview
This project is an interactive **Power BI dashboard** built using Superstore sales data.  
It provides insights into sales performance, profit trends, customer segments, and regional analysis to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-performing categories and regions
- Understand customer segment behavior
- Track business KPIs in a visual and interactive way

---

## 📁 Dataset Used
- Superstore Sales Dataset
- Columns include:
  - Order ID, Order Date, Ship Date
  - Segment, Category, Sub-Category
  - City, State, Region
  - Sales, Quantity, Discount, Profit

---

## 📊 Key KPIs (Cards)
- 💰 Total Sales
- 📈 Total Profit
- 🧾 Total Orders
- 📊 Profit Margin 

---

## 📌 Visualizations Used
- Clustered Column Chart (Sales & Profit comparison)
- Line + Column Chart (Trend analysis)
- Pie/Donut Charts (Segment & Category distribution)
- Map Visual (City/Region analysis)
- Slicers (Category, Segment, Region, Date)

---

## 🔍 Key Insights
- Which regions generate the highest profit
- Best and worst performing product categories
- Sales distribution across customer segments
- Impact of discounts on profit

---

## 🛠 Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning & Transformation

---

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Explore visuals using slicers and filters
